# Dog Kennel Business Model — GitHub Pages Site

## To host on GitHub Pages

1. Create a new GitHub repo (e.g. `dog-kennel-business-model`)
2. Push the contents of this `site/` folder to the `main` branch (or push the whole project and set Pages root to `/site`)
3. Go to repo Settings → Pages → Source: `main` branch, folder: `/site` (or `/root` if you moved files up)
4. GitHub will publish to: `https://yourusername.github.io/dog-kennel-business-model`

## To view locally right now

Open `site/index.html` directly in a browser — the Leaflet map and all tabs work without a server.

## Site Sections
- **Overview** — executive summary, key stats, risk/opportunity matrix
- **Competitor Map** — interactive Leaflet map with all competitors pinned
- **Competitor Intel** — detailed cards for each competitor
- **Market** — CAGR explained, demand drivers, capacity math
- **Financials** — capital stack, buildout costs, revenue model, unit economics
- **Operations** — daily schedule, staffing model, daycare vs. boarding
- **Regulations** — licensing path, GDA standards, zoning contacts
